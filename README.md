# Readme API Demo

A small demo site for [readme-api](https://www.github.com/masonlet/readme-api). It provides a simple interface for fetching and rendering repository README content through the API.

## Deployment & Configuration

### Prerequisites
- npm
- Node.js 18+

### 1. Clone the Repository
```bash
# Clone Readme API Demo
git clone https://github.com/masonlet/readme-api-demo.git
cd readme-api-demo
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run Locally
```bash
npm run dev    # http://localhost:5173
npm run build  # Production Vite build
npm run preview # Preview production build
```
